# PTTKHTTT
Yêu cầu đồ án
1. Team Information
 Danh sách, vai trò và công việc của từng thành viên trong nhóm
2. Problem statement.
 Mục đích đồ án (purpose)
 Phạm vi thực hiện (scope)
3. Software Requirements Specification
 Definition, Acronyms and Abbreviations (Các định nghĩa, từ viết tắt)
 Technologies to be used
 System functions (các chức năng hệ thống)
 User characteristics (xác định người dùng hệ thống)
 Performance Requirements
 Software requirements: reliability, security, maintainability, Portability
4. Project plan
 Lập tiến độ thực hiện đồ án bằng biểu đồ Grantt
 Xác định các mốc cho các phiên bản của hệ thống
5. Mô hình hóa yêu cầu : Use Case model, Activity diagram
6. Biểu đồ trình tự: Sequence diagrams.
7. Biểu đồ lớp chi tiết: Class Diagram
8. Thiết kế database
9. Thiết kế giao diện: User Interface
10.Deployment diagrams.
11.Kết luận, đánh giá hệ thống.
12.Hướng phát triển.
